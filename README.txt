This is a basic project using Flask and SQLalchemy.

There project consists of a microblog where every user can make a comment.
Users will see all comments from followed users on their main page,
but can check out all post from every user on the explore page.

There are basic concepts such as APIs, requet types, error handling, testing, authentication, templates usages.
An introduction to Bootstrap to make the site more beatiful.

There are a lot of dependencies that can be founf on the "dependencies.txt" file.

The db was not included for obvious reasons. To recreate it use:
  flask-sqlalchemy
  flask-migrate
